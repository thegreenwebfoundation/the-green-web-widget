This plugin adds The Green Web Foundation's green hosting badge as a WordPress widget.

== Description ==

This is a simple plugin which allows you to add [The Green Web Foundation](https://www.thegreenwebfoundation.org/)'s green hosting badge as a widget.

== Installation ==

1. Upload `the-green-web-widget` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Add The Green Web Widget to any widgetized area via Appearance -> Widgets or Appearance -> Customizer.

== Frequently Asked Questions ==

= How do I check if my website is hosted green? =

You can check if your website is hosted green at: [www.thegreenwebfoundation.org/](https://www.thegreenwebfoundation.org/) The widget will show your website as "grey" if you don't have a green host.

= How can I find a green host? =

Check out The Green Web Foundation's worldwide directory of green hosts at: [www.thegreenwebfoundation.org/directory/](https://www.thegreenwebfoundation.org/directory/)
